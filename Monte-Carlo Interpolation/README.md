# Интерполяция методом Монте-Карло и линейная интерполяция (C++)

## Обзор

В данном проекте реализована численная интерполяция функции двух переменных с использованием следующих методов:

- линейная интерполяция;
- интерполяция методом Монте-Карло.

Программа вычисляет значения функции на прямоугольной сетке и позволяет оценить эффективность используемых методов.

Время выполнения измеряется с использованием библиотеки **OpenMP**, **Intel Threading Building Blocks (TBB)**.

Проект предназначен для учебной работы по дисциплине **"Вычислительная физика"**.

---

## Математическая модель

Функция, используемая в эксперименте:

f(x,y) = (-1 + x + y)^2

Интерполяция выполняется на области:

x, y ∈ [0,1]

На этой области строится прямоугольная сетка узлов.

---

## Реализованные алгоритмы

### Линейная интерполяция

Точное значение интерполяции вычисляется по формуле:

z = (1-x)(1-y)f(0,0) + x(1-y)f(1,0) + (1-x)y f(0,1) + xy f(1,1)

Этапы алгоритма:

1. Генерация узлов сетки по координатам **x** и **y**.
2. Вычисление значения интерполяции для каждого узла.
3. Сохранение результатов в файл.

Выходной файл:

Linear.txt

Формат данных:

x    y    z

---

### Интерполяция методом Монте-Карло

Метод Монте-Карло выполняет приближённое вычисление значения функции с использованием случайных выборок.

Этапы алгоритма:

1. Генерация **N случайных выборок**.
2. Формирование случайных переменных, зависящих от координат сетки.
3. Вычисление значения функции для каждой выборки.
4. Вычисление среднего значения.

Приближение:

z(x_i,y_j) ≈ (1/N) Σ f(e1,e2)

Выходной файл:

MC.txt

Формат данных:

x    y    z

## Структура программы

main()
│
├── Linear_interpolation()
│      └── Linear_f()
│
├── MC_interpolation()
│
└── f()

### Функции

| Функция | Описание |
|-------|-----------|
| f(x,y) | вычисляет значение функции |
| Linear_f(x,y) | вычисляет билинейную интерполяцию |
| Linear_interpolation() | вычисляет значения интерполяции на сетке |
| MC_interpolation() | выполняет оценку методом Монте-Карло |

---

## Параметры

| Параметр | Описание | Значение |
|---------|-----------|----------|
| n1 | размер сетки по оси X | 26 |
| n2 | размер сетки по оси Y | 26 |
| N | количество выборок метода Монте-Карло | 1000 |

Шаг сетки:

h1 = 1 / (n1 - 1)  
h2 = 1 / (n2 - 1)

---

## Измерение производительности

Время выполнения измеряется с использованием:

tbb::tick_count

Пример вывода программы:

Linear interpolation time: 0.00012  
Monte Carlo interpolation time: 0.00345

Это позволяет сравнить эффективность реализованных алгоритмов.
