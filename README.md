# Проекты по численным методам и параллельным вычислениям (C++)

Данный репозиторий содержит несколько учебных проектов, посвящённых
численным методам, моделированию физических систем и параллельным
вычислениям на языке C++.

Цель проектов - исследование алгоритмов численного моделирования,
сравнение их вычислительной эффективности и применение библиотек
параллельного программирования для ускорения вычислений.

В репозитории представлены следующие проекты:

## 1. Интерполяция методом Монте-Карло
Программа реализует интерполяцию функции двух переменных на прямоугольной сетке 
с использованием линейной интерполяции и метода Монте-Карло. Вычисления выполняются 
для заданной области определения функции, а результаты сохраняются в файлы для 
последующего анализа и визуализации. В программе также производится измерение времени 
выполнения алгоритмов для сравнения их вычислительной эффективности.

## 2. Отображение Пуанкаре
Программа реализует интегрирование системы дифференциальных уравнений с использованием
метода Рунге–Кутты 4-го порядка и визуализирует траектории движения и отображения Пуанкаре.
Приложение позволяет задавать начальные условия, выбирать энергию системы и строить фазовые
проекции (x, y), (x, p_x) и (y, p_y). Графическая визуализация реализована с использованием
Windows Forms и отрисовывается в реальном времени.
