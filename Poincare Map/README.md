# Визуализация отображения Пуанкаре

## Описание проекта

Данный проект представляет собой приложение с графическим интерфейсом
(Windows Forms), написанное на **C++**, предназначенное для
численного моделирования нелинейной динамической системы и визуализации
её фазового пространства.

Программа выполняет численное интегрирование системы дифференциальных
уравнений с помощью **метода Рунге-Кутты 4-го порядка** и позволяет
визуализировать:

1. Траекторию движения частицы
2. Отображение Пуанкаре
3. Фазовые проекции системы
4. Границу потенциальной области движения

Графика выводится в реальном времени в графическую панель приложения.

------------------------------------------------------------------------

# Математическая модель

Динамика системы описывается следующей системой дифференциальных
уравнений:

dx/dt = p_x\
dy/dt = p_y

dp_x/dt = -x - 2xy\
dp_y/dt = y^2 - y - x^2

Система соответствует **нелинейной гамильтоновой системе**.

Начальное значение импульса (p_x) вычисляется из закона сохранения
энергии:

p_x = sqrt(2E - p_y^ - (x^2 + y^2) - 2x^2y + (2/3)y^3)

где

-   (x, y) --- координаты
-   (p_x, p_y) --- импульсы
-   \(E\) --- энергия системы

------------------------------------------------------------------------

# Численный метод

Для интегрирования системы используется **метод Рунге-Кутты 4-го
порядка**.

Алгоритм вычисления шага интегрирования:

1.  Вычисляется правая часть системы
2.  Производится четыре промежуточных вычисления
3.  Результаты усредняются для получения значения на следующем шаге

Метод обеспечивает хорошую точность при моделировании нелинейных систем.

------------------------------------------------------------------------

# Возможности программы

Программа позволяет пользователю:

-   задавать начальные условия:

    -   \(x\)
    -   \(y\)
    -   (p_y)

-   автоматически вычислять (p_x)

-   выбирать значение энергии системы:

    -   1/12
    -   1/8
    -   1/6

-   строить:

    **1. Траекторию (x, y)**\
    движение частицы в конфигурационном пространстве

    **2. Отображение Пуанкаре (y, p_y)**\
    точки пересечения траектории с выбранной поверхностью

    **3. Фазовую проекцию (x, p_x)**

-   отображать границу потенциальной области движения

-   запускать и останавливать численное моделирование

------------------------------------------------------------------------

# Интерфейс программы

Графический интерфейс включает:

### Ввод параметров

-   X --- начальная координата x\
-   Y --- начальная координата y\
-   P_Y --- начальный импульс по y\
-   выбор энергии системы

### Кнопки управления

-   **Initialization** --- инициализация параметров системы\
-   **Start** --- запуск моделирования\
-   **Stop** --- остановка расчёта\
-   **Pot** --- построение границы потенциала

### Панель визуализации

Отображает:

-   траектории
-   фазовые портреты
-   отображения Пуанкаре

------------------------------------------------------------------------

# Структура проекта

Проект состоит из следующих основных файлов:

**MyForm.cpp**\
точка входа программы и запуск графического интерфейса.

**MyForm.h**\
содержит реализацию интерфейса и основную логику программы.

**RKutta.cpp**\
реализация алгоритма интегрирования.

------------------------------------------------------------------------

# Выходные файлы

Во время работы программы могут создаваться файлы:

**res.txt**\
данные траектории системы.

**pot.txt**\
точки границы потенциальной области.

Эти файлы можно использовать для последующего анализа или построения
графиков в сторонних программах.

# Запуск программы

1.  Открыть проект в **Microsoft Visual Studio**
2.  Собрать проект
3.  Запустить приложение
4.  Ввести начальные условия
5.  Нажать **Initialization**
6.  Нажать **Start** для начала моделирования
